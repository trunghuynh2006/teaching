You have a Power Platform environment that contains a Microsoft Dataverse database.

A user named User1 plans to create a canvas app named App1 that connects to the Dataverse database. User1 will share App1 with other users.

You need to ensure that User1 can share App1. The solution must follow the principle of least privilege.

Which security role should you assign to User1?

Select only one answer.

The Environment Maker role is the least-privilege security role that allows a user to create apps, flows, and connections and to share apps with others in the environment. The System Customizer and System Administrator roles provide far broader permissions, including customization and administrative rights across the environment, which exceeds the requirements and violates least privilege. Delegated Admin applies to Microsoft partner-managed environments and is not required in this scenario.
