You have a Power Platform environment.

You plan to create a flow in Microsoft Power Automate by using a template.

You need to create a flow that posts a message to Microsoft Teams when a video is uploaded to Microsoft Stream. The flow must be created in the default solution and must minimize administrative effort.

Which type of flow should you create?

Select only one answer.

An automated cloud flow is the correct choice because it runs automatically in response to a trigger—in this case, when a video is uploaded to Microsoft Stream. This matches the requirement to minimize administrative effort since no manual intervention is needed once the flow is set up. An instant cloud flow requires manual execution, which would not meet the requirement. A scheduled cloud flow is designed to run at fixed intervals and is not triggered by events in other services. A business process flow is used in model-driven apps to guide users through predefined steps, not for automation between Stream and Teams.
