You have a legacy desktop application that must be automated by using Microsoft Power Automate for desktop.

The automation must read data from a file, and then enter the data into the legacy application.

You need to identify the correct sequence of actions for the desktop flow.

Which sequence of actions should you identify?

Select only one answer.

The correct sequence is to first extract the data from the file so that it is ready to be used, then launch and activate the legacy application to ensure it is available for input, and finally enter the data into the application. Launching the app before extracting data would leave the application idle while waiting, and attempting to enter data before the app is ready would fail. Extracting after input is illogical since the data must be available beforehand.
