You plan to create a flow using Power Automate. The flow will allow users to press a button in the Power Automate mobile app, which will send a reminder email notification.

You need to determine which type of flow to create.

Which type of flow should you create?

Select only one answer.

This item tests the candidate’s knowledge of the different types of triggers and how they are used in flows.

Instant flows can be triggered by a button in the Power Automate mobile app. A scheduled flow runs at a particular day and time with a specific frequency. In this case, the flow can happen whenever a person decides to run the flow, without any particular schedule. An automated flow runs after a particular event happens in another system. In this case, there is not another application or system that fires the event. Instead, a human is deciding to start the flow. Business process flows are used for guiding users through steps in a business process. However, business process flows are not initiated by pressing a button in the Power Automate mobile app.
