You have a Power Platform environment.

You need to create a canvas app that meets the following requirements:

  • Can be published in the Microsoft Teams app store
  • Connects to data from three different sources
  • Supports up to 1,000 concurrent users

What should you use?

Select only one answer.

The correct answer is Microsoft Power Apps Studio because this is the dedicated design surface used to build and configure canvas apps, including connecting to multiple data sources and preparing the app for publishing in Teams or other channels. Dataverse provides scalable storage and concurrency support but is not the tool used to create the app itself. The Power Apps mobile app is only for running apps on mobile devices, not for building them. The Power Apps maker portal is used to access environments and manage solutions, but the actual creation and customization of canvas apps takes place in Power Apps Studio.
