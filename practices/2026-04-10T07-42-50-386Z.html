You have a column that allows users to select one of two options: Approve or Reject.

This item tests the candidate’s knowledge of column types in Microsoft Dataverse. Identifying the appropriate column type for a requirement is a fundamental skill for designing a data model using Microsoft Dataverse.

The Choice field allows user to select a single value from a list of available values. User also have the option not to choose any value from the options. In that case the data for the column will be saved as null in data store. The Choices field allows a user to select more than one value from the list of available options. Approve and Reject options can be enabled using Yes/ No column type. However, it can have only two values. Text fields allow free user text without any data validations. It cannot be used to display options to user.
