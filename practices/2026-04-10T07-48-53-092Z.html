You have a Microsoft Excel file that contains employee expense information. The file is updated daily.

You need to create a cloud flow that runs automatically every time the file is updated.

What should you use to trigger the flow?

Select only one answer.

The correct answer is a Microsoft OneDrive trigger because it allows the flow to automatically run when the file stored in OneDrive is modified, which matches the requirement of reacting to file updates. A OneDrive action is used after a trigger to perform operations but cannot initiate the flow. A Power Apps action or trigger would require a user to manually start the flow from an app, which does not satisfy the requirement for automation on file updates.
