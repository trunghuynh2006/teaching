Forms can be included in a Power Pages site. External users can access Dataverse data by using these forms and can create, view, and edit rows in a Dataverse table. Power BI content can be embedded in a Power Pages site, but Power Pages does not create the visualizations; Power BI is responsible for the visualizations. Integrations between systems can be performed by using connectors in Power Automate cloud flows. Chatting between AI-powered bots and customers is performed by using Microsoft Copilot Studio.
